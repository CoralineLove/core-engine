// Project Overview
// ---------------
// The core-engine project is a software framework for building high-performance applications.
// It provides a set of reusable components and tools for building scalable, maintainable, and efficient software systems.

// Requirements
// ----------
// The following components are required to build and run the core-engine project:

// * C++11 compiler (or higher)
// * Boost C++ Libraries (version 1.68 or higher)
// * Google Test framework (version 1.8 or higher)

// Building the Project
// -------------------
// To build the project, navigate to the project root directory and run the following command:

// ```bash
// mkdir build
// cd build
// cmake ..
// make
// ```

// Running the Project
// ------------------
// After building the project, you can run the following command to execute the main application:

// ```bash
// ./core-engine
// ```
